# minetutrle API

## Создания чертежника

```Python
from mineturtle import *

turtle = Turtle()
```

`right(angle)`

`up(angle)`

`down(angle)`

`pendelay(x)`

`angle(x)`

`verticalangle(x)`

`angles(compass,vertical,roll)`

`penblock(x)`

`goto(x,y,z)`

`follow()`

`nofollow()`

`turtle(x)`

`penwidth(x)`

`pitch(x), yaw(x) and roll(x)`

`startface(), endface()`

`push(), pop()`

`gridalign()`
